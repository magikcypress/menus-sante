# Menus Santé

Site de menus de saison avec 135 recettes françaises, des quantités détaillées et un menu aléatoire généré chaque semaine.

**https://menu.rouquin.me**

## Fonctionnalités

- **4 saisons** : Hiver, Printemps, Été, Automne
- **135 recettes** avec ingrédients quantifiés pour 4 personnes
- **Menu hebdomadaire aléatoire** : change automatiquement chaque semaine
- **Curseur de portions** : ajuste les quantités de 1 à 12 personnes
- **Recherche par ingrédients** : sélectionner des produits de saison pour trouver des recettes
- **Filtrage par catégorie** : soupes, entrées, plats, accompagnements, desserts
- **Recette aléatoire** : bouton pour découvrir une recette au hasard
- **Produits de saison** : légumes, fruits, poissons, viandes

## Stack

- HTML + Tailwind CSS (CDN)
- JavaScript vanilla (aucune dépendance)
- Hébergé sur Vercel

## Lancer en local

Ouvrir `index.html` dans un navigateur.
